
### 一个基础的react项目结构

项目集成：
1. React,React-dom,react-router-dom,Redux,redux-thunk
2. webpack + gzip压缩
3. 默认压缩css js
4. loadable按需加载组件
5. react-hot-loaer
6. sass/css
7. 默认采用google ESlint
8. webpack-bundle-analyzer 可视化分析，端口9000

#### Installation
```
npm install
```

#### Usage:

##### development:
```
npm start 
（端口9200）
```
##### test-build:
```
npm run test-build
```
##### build:
```
npm run build
```
